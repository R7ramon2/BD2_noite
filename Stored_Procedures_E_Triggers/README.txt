-- O Primeiro Stored Procedure Pega os dados sobre o posto mais visitado, assim como o cliente que o mais visita e o numero de visitas

-- O Segundo Stored Procedure Pega o total de ganhos de um posto assim como uma serie de seus dados

-- O terceiro Encontra o posto mais barato de uma Cidade dado como parametro uma Cidade

Os Demais Stored Procedures estao relacionados aos triggers, onde (Nome_Do_Query_File)_Trigger é o Stored Procedure e (Nome_Do_Query_File)_Real_Trigger é o nome do seu respectivo Trigger

--As trasactions são referentes a inserção tanto na tabela de posto quanto na de posto_combustivel, onde caso os itens da tabela sejam inseridos corretamente ou erroneamente, uma mensagem é gerada no sistema.

-- A Stored Procedure pega uma lista decrescente dos carros que mais visitaram um posto dado o CNPJ desse posto.

-- A Function é acompanhada com a stored procedure.
